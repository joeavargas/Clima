# Clima ⛈
Weather app using OpenWeatherMap API endpoint

## SDK / Libraries / Frameworks
- UIFoundation
- UIKit
- CoreLocation

## 📲 Installation
Download the project to your local machine and open with Xcode or clone using the `https://github.com/joeavargas/Clima.git` link.
